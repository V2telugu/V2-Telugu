# V2 తెలుగు – సృష్టికర్తల ప్రతిజ్ఞ
## Creator Pledge

నేను V2 తెలుగు లో సృష్టికర్తగా చేరుతున్నప్పుడు,
కింది విలువలకు నేను కట్టుబడి ఉంటాను.

### నేను మాటిస్తాను
- నా కంటెంట్ నా స్వంత సృష్టి అవుతుంది
- తెలుగు భాషను గౌరవిస్తాను
- సమాజాన్ని బాధించే విషయాలు సృష్టించను
- నిజాయితీగా, బాధ్యతగా మాట్లాడుతాను
- వ్యూస్ కంటే విలువకు ప్రాధాన్యం ఇస్తాను

### నేను అంగీకరిస్తాను
- విమర్శను స్వీకరించడానికి
- కమ్యూనిటీ గైడ్‌లైన్స్ పాటించడానికి
- అవసరమైతే సవరణలు చేయడానికి

### V2 తెలుగు నుండి నాకు ఆశ
- గౌరవం
- పారదర్శకత
- న్యాయమైన వ్యవహారం
- నా స్వరానికి స్థానం
