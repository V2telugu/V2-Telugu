# V2 తెలుగు – చార్టర్
## వారి గుండె చప్పుడు

### మా లక్ష్యం (Our Purpose)
ప్రపంచవ్యాప్తంగా ఉన్న తెలుగు ప్రజల స్వరం
ఒకే వేదికపై, గౌరవంతో, స్వేచ్ఛగా వినిపించటం.

### ఎందుకు V2 తెలుగు?
వ్యూస్ కోసం కాదు.
వైరల్ కోసం కాదు.
డబ్బు కోసం కాదు.

మన భాష, మన సంస్కృతి, మన భావాలు
సజీవంగా ఉండాలి అనే బాధ్యత కోసం.

### మేము నమ్మేది
- సృష్టికర్తకు గౌరవం
- సమాజానికి ప్రాధాన్యం
- నాణ్యతకు విలువ
- పారదర్శకత

### మేము ఎప్పటికీ చేయము
- ద్వేషాన్ని ప్రోత్సహించము
- అబద్ధాలను వైరల్ చేయము
- అల్గోరిథమ్‌లకు మనుషులను బలిచేయము
- తెలుగు ఆత్మను అమ్ముకోము

### సృష్టికర్తలతో మా బంధం
V2 తెలుగు సృష్టికర్తలను వినియోగదారులుగా కాదు,
భాగస్వాములుగా చూస్తుంది.

### మా సంకల్పం
నెమ్మదిగా అయినా నిజంగా,
చిన్నదిగా అయినా గట్టిగా,
తెలుగు గుండె చప్పుడిగా నిలవడం.
