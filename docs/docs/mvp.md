# V2 తెలుగు – MVP
## Minimum Viable Platform

### MVP లక్ష్యం
తెలుగు సృష్టికర్తలకు
సురక్షితమైన, గౌరవప్రదమైన
కంటెంట్ పంచుకునే వేదిక ఇవ్వడం.

ఈ MVP ద్వారా:
- సృష్టికర్తలు కంటెంట్ అప్లోడ్ చేయగలగాలి
- ప్రేక్షకులు కంటెంట్ చూడగలగాలి
- ప్లాట్‌ఫామ్ నిజమైన అవసరాలు తెలుసుకోవాలి

---

## MVP లో ఉండే ఫీచర్లు

### 1. యూజర్ అకౌంట్లు
- Viewer (చూసేవారు)
- Creator (సృష్టికర్తలు)

### 2. సృష్టికర్త ప్రొఫైల్
- పేరు / ఛానల్ పేరు
- చిన్న పరిచయం
- కంటెంట్ రకం

### 3. కంటెంట్ అప్లోడ్
- వీడియో మాత్రమే
- పరిమిత వ్యవధి (ఉదా: 15 నిమిషాలు)
- మాన్యువల్ రివ్యూ అవసరం

### 4. వీక్షణ పేజీ (Watch Page)
- వీడియో ప్లేయర్
- శీర్షిక
- సృష్టికర్త పేరు

### 5. అడ్మిన్ రివ్యూ
- కంటెంట్ అప్రూవ్ / రిజెక్ట్
- గైడ్‌లైన్స్ ఆధారంగా నిర్ణయం
